# 🎯 Interactive Quiz Application

An interactive and dynamic quiz application built using **HTML, CSS, and JavaScript**.  
This project includes modern UI design, timer functionality, score tracking and downloadable certificate generation.
---
## 🚀 Features
- ✅ Dynamic Question loading
- ⏱ Countdown Timer for Each Question
- 🎯 Instant Feedback (Correct / Wrong)
- 📊 Score Calculation
- 🔄 Restart Without Page Reload
- 🎓 Professional Downloadable Certificate
- 💻 Fully Responsive UI
---
## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Canvas API (for Certificate Generation)
---

## 📸 How It Works
1. Click **Start Quiz**
2. Questions are displayed dynamically
3. Each question has a timer
4. Score is calculated automatically
5. Certificate can be downloaded after completion
